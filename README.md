## вЬ® Overview

Lumina renders content into the Windows desktop **`WorkerW`** layer, so your wallpaper
sits *behind the icons* but *above* the static desktop background вАФ a true live
wallpaper. It's built with **Electron**, runs entirely locally, and needs **no API keys**.

Every monitor is configured independently: its own source, scaling, color effects,
parallax, audio-reactivity, particle overlays, and info widgets.

## рЯФБ Playback & automation

- **Playlists** вАФ rotate multiple wallpapers on a timer (shuffle optional), with smooth **video crossfades**
- **Day/night scheduling** вАФ switch wallpapers by time of day
- **Night shift** вАФ automatic warm tint & dimming that tracks your **real local sunrise/sunset** (fixed-hours fallback offline)
- **Weather-reactive** вАФ live rain/snow overlay that matches your local weather
- **Auto-pause** вАФ behind fullscreen apps, on battery, when idle, or for **specific apps** you name
- **Power saving** вАФ cap wallpaper FPS, and a battery-saver that drops framerate & render scale when unplugged
- **Profiles** вАФ save & switch your whole multi-monitor setup; export/import as JSON
- **Global hotkeys & tray** вАФ pause-all (`Ctrl+Alt+P`), next wallpaper (`Ctrl+Alt+N`), per-display tray menu
- **Auto-update** вАФ packaged builds check GitHub Releases and update in the background
- **Self-healing** вАФ automatically re-attaches if `explorer.exe` / the desktop layer restarts
- **Mixed-DPI aware** вАФ correct per-monitor physical-pixel layout on multi-monitor setups

## рЯ§Ц AI depth maps

Adding a **2.5D depth** wallpaper without a depth map generates one automatically with
[MiDaS v2.1 small](https://github.com/isl-org/MiDaS) running **entirely on-device**
(ONNX Runtime, CPU вАФ inference takes well under a second). The ~64 MB model downloads
once on first use into the app's data folder; nothing ever leaves your machine.
Use the item's вЪЩ вЖТ *Regenerate depth map* after swapping in your own map to go back to AI.

## рЯІ© How it works

```
Progman вФАвФАвЦЇ WorkerW (desktop wallpaper host)
                вФФвФАвФА Lumina BrowserWindow  вЧДвФАвФА re-parented here (WS_CHILD)
                       вФФвФАвФА <video> / <canvas> / shader / web iframe
```

- Re-parents borderless Electron windows into the desktop `WorkerW` so content
  renders behind icons.
- Disables DirectComposition + forces device-scale-factor 1 so the GPU swap-chain
  presents correctly in the re-parented, mixed-DPI window.
- Win32 FFI via [`koffi`](https://koffi.dev) (`SetParent`, `SetWindowPos`, loopback audio, вА¶).

## вЪЩпЄП Tech & config

- **Electron 31** ¬Ј vanilla JS ¬Ј `koffi` for Win32 ¬Ј `yt-dlp` for YouTube ¬Ј `onnxruntime-node` + [MiDaS](https://github.com/isl-org/MiDaS) for on-device depth maps
