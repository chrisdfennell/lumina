# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**The short version:** be kind, be constructive, assume good faith. Harassment,
personal attacks, and NSFW content (including in shared presets) are not welcome
anywhere in this project — issues, PRs, discussions, or the gallery.

To report a problem, contact the maintainer privately via GitHub
([@chrisdfennell](https://github.com/chrisdfennell)) or use GitHub's built-in
report feature on the offending content. Reports are handled confidentially.
